
# 🌍 知識グラフ(Neo4j)とLLMを活用したGraphRAGベースの観光インテリジェント推薦システム

## 📋 プロジェクト概要
本プロジェクトは、知識グラフと大規模言語モデル（LLM）技術を基盤とした、インテリジェントな観光スポット推薦システムです。マイクロサービスアーキテクチャを採用し、知識グラフによる推薦アルゴリズムとLLMを統合することで、ユーザーにパーソナライズされた観光スポットの推薦サービスを提供します。システムは「ユーザー・観光スポット・カテゴリー・タグ」からなる知識グラフを構築し、ユーザーの類似性に基づく推薦、コンテンツの類似性に基づく推薦、対話型推薦など、多様なアプローチを実現しています。

### 🎯 プロジェクトの特色
* **知識グラフ駆動**: Neo4jグラフデータベースを使用して、複雑なエンティティ関係ネットワークを構築します。
* **マルチモーダル推薦**: ユーザーの類似性に基づく推薦、コンテンツに基づく推薦、対話型の推薦をサポートします。
* **スマート対話**: 大規模言語モデル（LLM）を統合し、自然言語による観光スポットの推薦に対応します。
* **リアルタイム可視化**: 知識グラフをインタラクティブに操作・閲覧できるUIを提供します。
* **地図連携**: 高徳地図（AMap）APIと連携し、観光スポットの位置情報を確認できます。
* **マイクロサービスアーキテクチャ**: フロントエンドとバックエンドを分離し、サービスを疎結合にすることで拡張性を高めています。
* **権限管理**: 管理者と一般ユーザーの2つのロールをサポートし、権限を分離しています。

## 🚀 主な機能

### 👤 ユーザー向け機能
* **ユーザー管理**: 新規登録、ログイン、個人情報の管理、パスワード変更。
* **観光スポット閲覧**: スポット一覧、詳細表示、検索・絞り込み、カテゴリー別閲覧。
* **パーソナライズ推薦**: ユーザーの行動に基づくインテリジェントな推薦（協調フィルタリング、コンテンツベース推薦）。
* **スマート対話**: AIアシスタントによる対話形式のスポット推薦。マルチターン対話に対応。
* **ユーザーアクション**: お気に入り登録、いいね、コメント、閲覧履歴、予約履歴。
* **マイページ（個人センター）**: お気に入りリスト、閲覧履歴、予約履歴、アカウント設定。
* **チャットシステム**: 複数セッションの管理、メッセージ履歴の保存、セッション削除。

### 🛡️ 管理者向け機能
* **データダッシュボード**: ユーザー数、スポット数、カテゴリー数、行動数、タグ数の統計。
* **ユーザー管理**: ユーザー情報の閲覧、ステータス管理、パスワードリセット、ユーザー削除。
* **スポット管理**: 観光スポットのCRUD操作、カテゴリー管理、タグ管理、地図表示。
* **知識グラフ管理**: データの同期、グラフの可視化、統計分析。
* **ユーザー行動分析**: ユーザーの行動履歴のトラッキングと確認。

### 🧠 コア推薦機能
* **協調フィルタリング**: ユーザー間の類似性に基づく推薦アルゴリズム。
* **コンテンツベース推薦**: 観光スポットのコンテンツ（属性）の類似性に基づくアルゴリズム。
* **対話型推薦**: 自然言語によるチャットを通じたインテリジェントな推薦。
* **グラフ検索拡張（GraphRAG）**: 知識グラフを活用し、LLMのハルシネーションを防ぎつつ、より高精度な推薦を実現。

## 🔧 コア機能の実装詳細
1.  **ユーザー認証と権限管理**: JWTを利用したトークン認証、管理者と一般ユーザーのRBAC（ロールベースアクセス制御）、フロントエンドのルートガード、バックエンドのAPIインターセプターを実装。
2.  **知識グラフの構築**: MySQLからNeo4jへの自動データ同期、エンティティ（ユーザー・スポット・カテゴリー・タグ）間の関係モデリング、Cypher言語による複雑なグラフクエリ、可視化UIを実装。
3.  **推薦アルゴリズム**: ユーザー類似度（協調フィルタリング）、コンテンツ類似度、ユーザーの会話からのキーワード抽出を利用したハイブリッドな推薦アプローチ。
4.  **インテリジェント対話システム**: 複数セッション対応、メッセージ履歴の永続化、複数LLMのサポート、自然言語によるおすすめスポットの提案。
5.  **ユーザー行動分析**: 閲覧、お気に入り、いいね、コメント、予約などの行動を記録・統計し、トレンドや人気の分析を実施。

## 🔮 技術的なハイライト
* **グラフ検索拡張 (GraphRAG)**: 知識グラフと大規模言語モデル（LLM）を組み合わせた最先端のイノベーション技術。
* **マルチモーダル推薦**: 協調フィルタリング、コンテンツベース推薦、対話型推薦の有機的な統合。
* **リアルタイムデータ同期**: MySQLからNeo4jデータベースへの遅延のないデータ同期メカニズム。
* **スマートキーワード抽出**: ユーザーの自然言語対話から、文脈を理解して観光スポットのキーワードをインテリジェントに抽出。
* **インタラクティブな可視化**: 知識グラフの直感的な表示・操作。
